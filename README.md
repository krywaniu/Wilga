# Wilga
Pascal browser liblary for easy games



Kod gry piszesz w Pascalu i kompilujesz do JS przez pas2js.
Wymaga: pas2js, pliku index.html, rtl.js i HTML5 Canvas.
Kompilacja: pas2js -Jc -O2 -Tbrowser main.pas
index.html musi załadować rtl.js oraz wygenerowany main.js.

Nie uruchamiaj bezpośrednio przez index.html (WAŻNE)
